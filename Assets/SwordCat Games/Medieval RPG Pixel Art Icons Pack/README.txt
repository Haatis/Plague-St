Thank you for downloading our asset!

This is a SwordCat Games production. Feel free to send requests with ideas for new icons to be added.
Contact email: swordcatgames@gmail.com

This asset is simple to use, the icons are all in the "Frames & Icons" folder, just need to pick them from there and use them.

Besides the icons there's also frames included to be used as background for the icons, choose the one you like!

If you're not going to use the included frames it's recommended that you use the icons with the same background color present in the frames, which is #3E3546, since they were designed using this background color.
